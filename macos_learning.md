# macOS_Learning

## shortcut keys

### 不熟练

`fn+f`: 全屏，对一些应用程序没用，改用`ctrl+cmd+f`

`option+cmd+space`: 在finder中搜索

`fn+backspace`: delete 

`cmd+\+`: 某些应用(terminal)字体变大
`cmd+\-`: 某些应用字体变小

### 已掌握

`cmd+c`, `cmd+x`, `cmd+v`

`cmd+q`: 关闭当前应用
`cmd+m`: 隐藏当前应用
`cmd+w`: 关闭当前窗口

`cmd+space`: 搜索

`f11`: 回到桌面
