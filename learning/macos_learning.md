# macOS_Learning

## shortcut keys

### 不熟练

`option+cmd+space`: 在finder中搜索

`双指+option`: 复制文件路径

`f11`: 回到桌面（对全屏模式没用）


`cmd+opt+d`: 显示/隐藏程序坞（对全屏模式没用）

### 已掌握

`cmd+c`, `cmd+x`, `cmd+v`

`cmd+q`: 关闭当前应用
`cmd+m`: 隐藏当前应用
`cmd+w`: 关闭当前窗口
`cmd+h`: 隐藏当前窗口

`cmd+\+`: 某些应用(terminal)字体变大
`cmd+\-`: 某些应用字体变小

`cmd+左右`: 行头或行尾
`opt+backspace`: 删除一个词

`cmd+space`: 搜索

`ctrl+shift+345`: 截图

`fn+backspace`: delete 
`fn+f`: 全屏，对一些应用程序没用，改用`ctrl+cmd+f`
